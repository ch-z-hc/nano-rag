\datedsubsection{\textbf{nano-rag：本地知识库 RAG 问答系统}}{2026年5月 -- 至今}
\role{Python, ChromaDB, DeepSeek API, aiohttp}{个人项目 \textperiodcentered\ \href{https://github.com/ch-z-hc/nano-rag}{GitHub}}
\begin{onehalfspacing}
\begin{itemize}
    \item 从零实现 RAG（检索增强生成）管线：文档解析（PDF/Markdown/代码）$\to$ 语义分块 $\to$
      向量化入库（ChromaDB + sentence-transformers）$\to$ 检索 $\to$ LLM 生成回答；全链路可本地运行
    \item 设计并实现混合检索与重排策略：向量语义检索 + BM25 关键词检索双路召回，RRF 融合排序，
      显著提升中英文混合知识库的检索精度
    \item 集成微信个人机器人通道：基于 ilinkai 协议实现扫码登录、长轮询消息同步、上下文感知回复；
      同时提供 Web 前端（aiohttp + Vanilla JS），支持 Markdown/LaTeX 公式渲染、拖拽上传、知识库管理
    \item 采用 DeepSeek API 作为生成模型，通过结构化 Prompt 控制输出质量（数学公式格式、中英文排版间距）；
      系统解耦为 store / ingest / query / channels 四层，支持多通道扩展
\end{itemize}
\end{onehalfspacing}
